## Project Description
This project implements a PERT (Program Evaluation and Review Technique) based scheduler using Python and Tkinter GUI.

It allows users to:
- Input tasks with time estimates (O, M, P)
- Define dependencies between tasks
- Automatically compute scheduling metrics
- Visualize results using tables, Gantt charts, and network diagrams

## Features
- Expected Time Calculation (PERT)
- Critical Path Detection
- Slack Calculation
- Project Duration
- Minimum Workers Required
- Task Assignment Scheduling
- Gantt Chart Visualization
- Network Diagram Visualization
- Save/Load Project Data (JSON)
